# 📊 臺北市 1999 派工資料互動儀表板  
Taipei City 1999 Dispatch Dashboard

一個以 **臺北市政府 1999 派工公開資料**為基礎的互動式前端資料儀表板，整合地理視覺化、統計圖表與多條件檢索，協助快速觀察城市問題的分布與熱點。

本專案為 **純前端、單一 HTML 檔案**，可直接以瀏覽器開啟，無需後端與安裝環境。

---

## 🔍 專案特色

- **即時資料串接**
  - 串接臺北市政府開放資料 API
  - 具備 CORS Proxy 與失敗重試機制
  - API 失效時自動切換為樣本展示模式

- **互動式資料視覺化**
  - 行政區案件量長條圖（可點擊反向篩選）
  - 前十大派工項目圓環圖
  - 七大核心派工類別比例分析

- **地理空間整合**
  - 內嵌 Google My Maps 派工總覽
  - 點擊案件列自動清洗地址並定位至 Google Maps

- **多條件資料檢索**
  - 行政區複選
  - 派工類別複選
  - 關鍵字搜尋（案件編號／地址／派工內容）
  - 圖表與篩選條件雙向互動

- **良好使用者體驗**
  - Tailwind CSS 響應式介面
  - 即時狀態提示與 Loading 視覺
  - 可動態載入更多資料筆數

---
