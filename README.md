# 📊 臺北市 1999 派工資料互動儀表板  
Taipei City 1999 Dispatch Dashboard

本專案為一個以 **臺北市 1999 市民熱線派工資料** 為基礎的互動式資料視覺化網站，目的在於協助使用者快速理解不同類型案件的分布情形、空間熱點與時間趨勢，作為資料分析、研究或展示之用。

> ⚠️ 本專案為 **非官方** 之資料視覺化展示，實際數據仍應以臺北市政府官方公告為準。

---

## ✨ 專案特色

- 📍 **地圖視覺化**：以地圖方式呈現 1999 派工案件的空間分布  
- 🔥 **熱點分析**：快速辨識案件集中區域  
- 🗂 **案件分類**：依派工類型進行篩選與比較  
- ⏱ **時間觀察**：觀察不同時間區段的案件變化  
- 🌐 **前端即時互動**：無需後端即可操作與瀏覽

---

## 🛠 技術架構

- 前端框架：HTML / CSS / JavaScript  
- 視覺化套件：D3.js（或對應使用之視覺化函式庫）  
- 地圖工具：Leaflet（或專案中實際使用之地圖套件）  
- 資料來源：臺北市政府開放資料平台（1999 派工相關資料）

## 🚀 使用方式

1. 下載或 clone 本專案
   ```bash
   git clone https://github.com/LynxTC/taipei-1999-dashboard.git
   ```
2. 直接以瀏覽器開啟 `index.html`
3. 透過地圖與互動元件探索派工資料

> 若遇到瀏覽器安全性限制（如本地端讀取檔案），建議使用簡易伺服器：
```bash
python -m http.server
```
